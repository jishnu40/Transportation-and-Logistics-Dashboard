## 📊 Project Overview
This project focuses on analyzing and visualizing key logistics metrics using **Power BI**.  
It provides a 360° view of delivery performance, vehicle & driver utilization, and customer insights, helping logistics teams make data-driven operational decisions.

The dashboard is designed with a **professional dark theme (#000000)** and **highlight color (#20E2E7)** for a clean and consistent analytical experience.

---

## 🎯 Key Insights & Dashboards
### 1️⃣ **Home Page**
- Custom-designed landing page with navigation buttons and icons.
- Serves as a starting point to explore different dashboards.

### 2️⃣ **Overview Dashboard**
- Highlights total shipments, delays, and performance KPIs.
- Provides an at-a-glance summary of logistics health.

### 3️⃣ **Delivery Performance Dashboard**
- Tracks on-time vs delayed deliveries.
- Shows delivery trends by date, route, and vehicle.

### 4️⃣ **Customer Dashboard**
- Analyzes customer shipment frequency and delivery success.
- Identifies top and bottom performing customers.

### 5️⃣ **Vehicles & Drivers Dashboard**
- Monitors truck utilization, mileage, and driver experience.
- Identifies top-performing vehicles and drivers.

### 6️⃣ **Conclusion Dashboard**
- Summarizes key metrics and operational takeaways.
- Provides visual performance trends for quick reporting.
